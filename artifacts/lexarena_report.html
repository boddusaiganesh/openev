Cognitive Dimension Profile
Tier Architecture
T1
Clause Reading 15%

Can the agent extract the exact sentence answering a legal question?
Metric: F2 score (recall-weighted) + Jaccard + Laziness Rate.
Data: CUAD dataset — 510 real contracts, 41 question types.

T2
Risk Classification 15%

Can the agent classify clause type, risk level, and recommend an action?
Metric: Weighted label accuracy across 3 difficulty tasks.
Data: OpenEnv tasks 1-3.

T3
Dependency Mapping 20%

Can the agent proactively map all hidden cross-contract dependency edges?
Metric: Precision/recall/F1 against ground-truth edge list.
Data: LexDomino scenario dependency_edges arrays.

T4
Crisis Easy 12.5%

Can the agent manage a single-contract crisis over 15 days?
Metric: Cash survival ratio + deadlines met + edges discovered.

T5
Crisis Medium 17.5%

Can the agent handle a multi-contract crisis with hidden dependencies?
Metric: Cash survival ratio — adversarial counterparties.

T6
Crisis Hard 20%

Can the agent survive 30-day full systemic cascade with compound shocks?
Metric: Cash survival — bankruptcy = 0 score.

Legal IQ Scoring Formula
Legal_IQ = 0.15 × T1_Reading_Score
           + 0.15 × T2_Classification_Score
           + 0.20 × T3_Dependency_Score
           + 0.50 × T4_T5_T6_Crisis_Score

where T4_T5_T6_Crisis_Score = (0.25×T4 + 0.35×T5 + 0.40×T6)

T1_score = 0.60×F2 + 0.25×Jaccard_mean + 0.15×(1 − laziness_rate)
T3_score = 0.50×recall + 0.25×precision + 0.15×edge_type_accuracy + 0.10×severity_order
LexArena v1.0 • Pure deterministic scoring • No LLM judges • Built on OpenEnv + LexDomino + ContractEval/CUAD
